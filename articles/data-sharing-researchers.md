# 🔬 Open Science and Data Sharing: Module 2

This module covers how to make your research data FAIR (Findable, Accessible, Interoperable, and Reusable) and how to select the right repository for publication.

## 📋 Learning Objectives
* Understand the core principles of **FAIR Data**.
* Learn how to license your datasets using Creative Commons (e.g., CC-BY).
* Evaluate different domain-specific repositories versus general ones (like Zenodo).

## 🛠️ Practical Checklist
- [ ] Check your target journal's specific data-sharing mandates.
- [ ] Select an appropriate open-access repository for your field of study.
- [ ] Apply a clear license text file (`LICENSE.txt`) to your dataset root folder.
