# 💼 Risk Management and Data Security: Module 2

This module shifts focus toward mitigating risks, handling sensitive data (PII), and implementing cloud security strategies at a team level.

## 📋 Learning Objectives
* Identify **Personally Identifiable Information (PII)** within team workflows.
* Implement encryption standards for data at rest and in transit.
* Establish a clear data retention and deletion schedule.

## 🛠️ Practical Checklist
- [ ] Audit your team's storage to ensure no unencrypted PII is stored on local desktops.
- [ ] Draft a 5-step emergency checklist for reporting potential data breaches.
- [ ] Set up automated 30-day archiving policies for inactive project folders.
