# 🔬 Data Management for Researchers: Module 1

Welcome to your first step in research data mastery. This module focuses on structuring and documenting your raw data to meet peer-review standards and institutional open-science policies.

## 📋 Learning Objectives
* Learn the **Folder Structure Best Practices** for reproducible research.
* Understand how to generate a comprehensive `README.txt` metadata file.
* Prep data files for long-term archiving in open repositories.

## 🛠️ Practical Checklist
Complete these tasks before moving to the next module:
- [ ] Create a dedicated `/data/raw/` folder that remains strictly read-only.
- [ ] Set up a standardized file naming convention (e.g., `YYYYMMDD_Project_DataType_vX`).
- [ ] Draft a data dictionary explaining all variables and column headers.

> **Important Note:** Never modify your raw data files directly. Always create a copy in a `/data/processed/` directory before starting your analysis.
